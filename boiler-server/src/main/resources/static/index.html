12. 信用分与资质规则

  • 新注册 Register用户默认信用分为 75。
  • 管理员 Admin可将信用分调整到 0-100 范围。
  • 信用分统一按规则计算：信息完善 40、互评基础 15、交易行为基础 10、平台合规 10。
  • 互评每个好评 Positive +3、每个差评 Negative -5；交易每完成一次 +2。
  • 已审核卖家 Seller更新资质资料后，资质状态会自动回到 PENDING。
  • 资质文件支持 BUSINESS_LICENSE 和 LEGAL_PERSON_ID 两类上传。
