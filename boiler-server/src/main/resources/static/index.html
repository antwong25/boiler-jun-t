12. 信用分与资质规则

  • 新注册用户默认信用分为 60。
  • 管理员可将信用分调整到 0-100 范围。
  • 卖家资质审核通过后，信用分会自动提升到不低于 80。
  • 已审核卖家更新资质资料后，资质状态会自动回到 PENDING。
  • 资质文件支持 BUSINESS_LICENSE 和 LEGAL_PERSON_ID 两类上传。
